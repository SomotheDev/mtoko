# Gymshark Design Research

## Key Design Elements

### Color Scheme
- **Primary Colors**: Black, White, Grey tones
- **Accent Colors**: Burgundy/Rich Maroon (seasonal), Athletic colors
- **Background**: Clean white/light grey backgrounds with dark text
- **Product highlights**: Uses color overlays and gradients on hero images

### Typography
- **Headers**: Bold, uppercase sans-serif fonts for impact
- **Body**: Clean, readable sans-serif
- **Product titles**: Mix of uppercase and sentence case
- **Style**: Modern, athletic, bold

### Navigation Structure
- **Top Navigation**: Horizontal menu with Women, Men, Accessories
- **Header Icons**: Search, Wishlist/Heart, Account, Shopping Cart
- **Announcement Bar**: Promotional messages at the very top
- **Logo**: Centered in header, bold GYMSHARK wordmark

### Layout Patterns
- **Hero Section**: Full-width image/video with overlay text and CTAs
- **Product Grids**: Clean grid layout with product cards
- **Product Cards**: 
  - Large product image
  - Product name/description
  - Price prominently displayed
  - Tags (e.g., "new", "oversized fit")
  - Hover effects for interaction
- **Sections**: Clearly defined sections with headers (e.g., "NEW IN", "POPULAR RIGHT NOW")
- **Call-to-Actions**: "View All" buttons, "Shop New In" links

### UI Patterns
- **Product Display**: Grid-based layout with consistent spacing
- **Category Sections**: Organized by gender and product type
- **Featured Collections**: Carousel/slider format for new arrivals
- **Footer**: Comprehensive links organized by categories
- **Modal/Popup**: Cookie consent with clear CTAs

### Visual Style
- **Photography**: High-quality lifestyle and product photography
- **Imagery**: Athletic models, gym environments, product close-ups
- **Spacing**: Generous whitespace, clean and uncluttered
- **Buttons**: Bold, high-contrast buttons with clear labels
- **Icons**: Simple, minimal line icons

### Key Features
- Product filtering and categorization
- Wishlist/favorites functionality
- User account system
- Shopping cart
- Search functionality
- Responsive design
- Product recommendations ("Welcome to your edit")

### Design Philosophy
- Clean, modern, athletic aesthetic
- Focus on product imagery
- Bold typography and clear hierarchy
- High contrast for readability
- Mobile-first responsive design
- Performance-focused (fast loading)
